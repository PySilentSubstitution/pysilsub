Welcome to *PySilSub*'s documentation!
======================================

*PySilSub* is a Python software for performing the method of silent substitution. See also `PyPlr <https://pyplr.github.io/cvd_pupillometry/index.html#>`_, a sister project which offers a Python framework for researching the pupillary light reflex with the `Pupil Core <https://pupil-labs.com/products/core/>`_ eye tracking platform.
