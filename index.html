Welcome to PySilSub’s documentation!¶

PySilSub is a Python software for performing the method of silent substitution. See also PyPlr, a sister project which offers a Python framework for researching the pupillary light reflex with the Pupil Core eye tracking platform.

Important notice¶

Please be aware that these documentation pages and the associated software are currently in the early stages of development.
